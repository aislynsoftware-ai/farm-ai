# Product Development

AI-powered product development platform containing:

- Plant Disease Detection
- Food Identification
- Plant Identification
- YOLO-based Inference
- Product Development APIs
- Image Processing
- Agricultural AI Models

## Features

- Disease Classification
- Crop Detection
- Prediction Image Generation
- User Authentication
- Coin-based Prediction System
- Admin Dashboard

## Tech Stack

- Python
- Flask
- YOLO
- MySQL
- Docker
- Hugging Face Spaces

## Deployment

This project is deployed using Docker on Hugging Face Spaces.
